# Приложение для умножения чисел

Простое веб-приложение на Golang с использованием Gin и MongoDB для умножения чисел и сохранения результатов.

## Функциональность

- Веб-интерфейс для ввода двух чисел
- Умножение чисел и сохранение результата в MongoDB
- Отображение истории результатов в таблице

## Технологии

- Golang
- Gin (веб-фреймворк)
- MongoDB (NoSQL база данных)
- Docker & Docker Compose (для запуска MongoDB)
- HTML/CSS

## Требования

- Go (версия 1.16+)
- Docker и Docker Compose

## Установка и запуск

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/yourusername/go_multiply_app.git
   cd go_multiply_app
   ```

2. Запустите MongoDB с помощью Docker Compose:
   ```
   docker-compose up -d
   ```

3. Запустите приложение:
   ```
   go run main.go
   ```

4. Откройте в браузере:
   ```
   http://localhost:8080
   ```

## Структура проекта

- `main.go` - основной файл приложения
- `models/result.go` - модель данных для результатов
- `templates/index.html` - HTML шаблон интерфейса
- `docker-compose.yml` - конфигурация Docker для MongoDB

## Как использовать

1. Введите два числа в соответствующие поля
2. Нажмите кнопку "Умножить"
3. Результат будет сохранен в базе данных и отображен в таблице истории
